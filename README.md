# Responsabilidade Social na Tecnologia da Informação

Esta apresentação trata da responsabilidade social dos profissionais da
Tecnologia da Informação.

Este trabalho foi desenvolvido para a disciplina de **Meio Ambiente, Sociedade e
Tecnologia**, do professor **Volmir Fontana**, no curso de **Bacharelado em
Engenharia de Software**, da **UNIVILLE** (*Universidade da Região de Joinville*).

Trabalho desenvolvido por:

- Vanderlei Sbaraini Amancio
- Geferson Eurico dos Santos
- Everson Bittencourt

- - -

Esta apresentação utiliza o framework [Electron](http://electron.atom.io/), do
*Github*, para desenvolvimento da "aplicação". Para desenvolvimento da apresentação,
foi utilizado a biblioteca [Bespoke](http://markdalgleish.com/projects/bespoke.js/).
